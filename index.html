Perguntas Frequentes

O que é JavaScript?

JavaScript é uma linguagem de programação usada por desenvolvedores
para fazer páginas interativas da Internet.

O que é HTML?

Sigla para HyperText Markup Language — Linguagem de Marcação de Hipertexto —, o HTML é o componente base da web. Isso quer dizer que ele permite a construção de websites e a inserção de novos conteúdos,
como imagens e vídeos, por meio dos hipertextos.

O que é CSS?

Você sabe o que é CSS? Trata-se de uma linguagem de marcação, amplamente utilizada com HTML ou XHTML, e representa diversas possibilidades para a formatação. O CSS ajuda a editar, alinhar,
remover e trabalhar no espaço entre elementos de uma página.
